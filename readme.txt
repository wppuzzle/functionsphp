== Description ==

To add HTML code counters and scripts — use examples in plugin code. Simply inserted HTML code in plugin will be displayed incorrectly and can lead to site malfunction.



== Installation ==

= Automatic installation from archive =

1. In console WordPress go to menu Plugins -> Add new.
1. At top of page, next to inscription Add plugins, click Upload new.
1. In appeared window click Choose file and select saved archive with plugin on computer. Then click button Install.
1. After plugin installation, system offers to activate it — click link Activate plugin
